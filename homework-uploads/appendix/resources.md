### Open Source Repositories

- **AWS ASL Avatar Generator (speech/text → gloss → avatar video):**  
  https://github.com/aws-samples/genai-asl-avatar-generator  
  *Uses LLM for glossing + stitches pre-segmented sign/pose clips into video; full infra (frontend, backend, data prep) runs on AWS.*

- **HamNoSys → SiGML Converter (MIT Licensed) + Unity C# Bridge:**  
  https://github.com/carolNeves/HamNoSys2SiGML  
  *Runs Python converter from Unity and outputs SiGML files to Assets.*

- **Text → Gloss → Pose → Video Pipeline:**  
  https://github.com/sign-language-processing/spoken-to-signed-translation  
  *Reference end-to-end system we can study or replicate.*

- **Sign-Kit (speech/text → 3D avatar gestures):**  
  https://github.com/spectre900/Sign-Kit-An-Avatar-based-ISL-Toolkit  

- **Arabic/Algerian Sign Language Avatar Translator:**  
  https://github.com/linuxscout/algerianSignLanguage-avatar?utm_source=chatgpt.com  
  *Shows dictionary, grammar rules, and animation assembly.*

- **English → ASL Translator with 2D Avatar:**  
  https://github.com/kevinjosethomas/sign-language-processing  

- **SignWave (ASL-related project):**  
  https://github.com/tan-ad/SignWave
