# Project Expenses

## Summary

AiSL was developed using free, open-source, and publicly available tools throughout the project. The one exception was a small purchase of OpenAI API credits to support development and testing of the gloss generation pipeline.

---

## Cost Breakdown

| Category | Description | Cost |
|---|---|---|
| AI/LLM API | OpenAI API credits (gpt-4o-mini) | $5.00 |
| Hardware | None | $0 |
| Software | Open-source and free-tier tools only | $0 |
| Cloud/Hosting | Free and student-provided infrastructure | $0 |
| Design Tools | Figma free tier | $0 |
| Miscellaneous | None | $0 |

**Total Spent: $5.00**

---

## No-Cost Resources

- **OpenAI API (gpt-4o-mini)** - Drove the constrained ASL gloss generation pipeline. $5 in credits covered the full scope of development and testing.
- **Unity (Personal License)** - Built the 3D avatar scene and WebGL export.
- **Studio Galt Sign Language Mocap Archive** - Sign pose and animation assets covering 1,915 signs.
- **Microsoft Rocketbox** - Avatar rig and base assets for signing playback.
- **WLASL Dataset (Kaggle)** - Word-level ASL video dataset with 2,000 clips.
- **React, Vite, Flask** - Frontend and backend frameworks.
- **VS Code** - Development environment.

---

## Notes

The $5 API spend was the only out-of-pocket cost for the entire project. Prioritizing free and open tools kept the project accessible and low-barrier, which aligns with the broader goal of making ASL learning technology available without cost constraints.
