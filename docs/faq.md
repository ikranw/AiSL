# FAQ

**1. Who is this for?**  
Beginner and intermediate ASL learners.

**2. Is this word-for-word translation?**  
No. ASL grammar is different from English.

**3. Can this replace an ASL teacher?**  
No. It is a practice tool.

**4. Is this a live interpreter?**  
No.

**5. Are there other language options**  
No, English is the only language offered at the moment.
