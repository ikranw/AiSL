# User Manual

## How It Works
1. You type an English sentence  
2. The system converts it to the correct ASL grammar  
3. A 3D avatar signs the sentence  

---

## Features
- ASL grammar conversion  
- 3D signing avatar  
- Playback controls (pause, replay, speed)  

---

## Limits
- Vocabulary depends on available signs  
- Complex phrases may not translate perfectly
